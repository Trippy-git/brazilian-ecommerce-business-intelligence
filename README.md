An end-to-end **Business Intelligence** project built using **MySQL** and **Power BI** to analyze over **100,000 Brazilian e-commerce orders** from the Olist Marketplace dataset.

The project transforms raw transactional data into actionable business insights through advanced SQL analysis, customer segmentation, operational performance monitoring, geographic analysis, and interactive Power BI dashboards.

---

# 📌 Project Overview

This project simulates a real-world Business Intelligence solution by analyzing transactional e-commerce data from the Brazilian Olist Marketplace.

The analysis begins with SQL-based data exploration and business analysis before progressing into interactive Power BI dashboards designed for executive decision-making.

The project focuses on four key business areas:

- Executive Sales Performance
- Operational Performance
- Customer Analytics
- Geographic Performance

The complete analytics workflow includes:

- Data Preparation
- Advanced SQL Analysis
- Customer Segmentation (RFM)
- Pareto Analysis
- Churn Risk Analysis
- Cohort Analysis
- Power BI Dashboard Development
- Business Insights & Recommendations

---

# 🎯 Business Problem

Modern e-commerce businesses generate millions of transactional records.

Without proper analysis, decision-makers struggle to answer important business questions such as:

- Which product categories generate the highest revenue?
- Which sellers perform best?
- Which customers are most valuable?
- Which customers are at risk of churn?
- Which regions generate the highest sales?
- Where are delivery bottlenecks occurring?
- Does the business follow the Pareto Principle (80/20 Rule)?

This project was designed to answer these questions through SQL analysis and interactive dashboards.

---

# 🎯 Project Objectives

- Analyze monthly sales and revenue trends.
- Identify top-performing product categories.
- Measure seller operational performance.
- Segment customers using RFM Analysis.
- Perform Customer Churn Analysis.
- Validate the Pareto Principle.
- Analyze delivery performance across Brazilian states.
- Build executive-level dashboards for business stakeholders.
